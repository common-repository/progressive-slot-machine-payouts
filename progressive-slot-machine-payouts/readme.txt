== Description ==

Displays the name and current payouts for several popular progressive poker-based slot machines. Companies include RTG, Microgaming and Play Tech.

== Installation ==

1. Upload [progressive_slot_machine_payouts.php] to your [/wp-content/plugins/] directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Click 'Appearance' -> 'Widgets'.
4. Drag the "Progressive Slot Machine Payouts" widget to your sidebar.
